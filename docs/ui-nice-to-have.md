# UI Nice-to-Have Features

> Features that would enhance the chat UI but are not critical for MVP.

## High Value / Low Effort

| Feature | Description | Effort |
|---------|-------------|--------|
| **Chat history persistence** | Save chat to localStorage, restore on page refresh | Low |
| **Copy answer button** | One-click copy answer text to clipboard | Low |
| **Export chat as markdown** | Download conversation for Apple Notes | Low |

## Medium Value

| Feature | Description | Effort |
|---------|-------------|--------|
| **Dark mode** | Toggle between light/dark themes | Low |
| **Mobile responsive** | Better layout on phone/tablet screens | Medium |
| **Clear chat button** | Reset conversation and start fresh | Low |

## Low Priority / Polish

| Feature | Description | Effort |
|---------|-------------|--------|
| **Typing indicator** | Animated dots while waiting for response | Low |
| **Syntax highlighting** | Highlight code blocks in answers (if any) | Low |
| **Keyboard shortcuts help** | Show available shortcuts (?, Cmd+K, etc.) | Low |
| **Answer feedback** | Thumbs up/down on responses (for future eval) | Medium |

## Out of Scope

- **Authentication** — Local project only, not needed
- **SSR/SEO** — SPA is fine for personal tool
- **Multi-user support** — Single user local app
